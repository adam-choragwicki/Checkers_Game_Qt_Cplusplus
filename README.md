# Checkers
The project is my implementation of classic Checkers player vs player game. It features standard rules:
* capture is mandatory
* any backward direction, including capture, is possible only for promoted pieces
* promoted piece can move in any direction, 1 field per turn

Project was unit-tested using Google Test framework. 

See screenshot.jpg for quick glance at game window.

Run Checkers/executable/Checkers.exe to start the game.
